## 5. 整体审美偏好

### 总体风格描述

```text
例如：顶会论文风格，现代、精致、信息丰富，但不能像商业宣传海报。
```

### 喜欢的风格

- [ ] 学术扁平插画
- [ ] 简洁矢量风格
- [ ] 轻量2.5D
- [ ] 写实科研插画
- [ ] 手绘或草图风格
- [ ] 信息图风格
- [ ] 极简风格
- [ ] 高密度系统图
- [ ] 其他：

### 不喜欢或禁止的风格

- [ ] 卡通化过强
- [ ] 商业宣传海报
- [ ] 科幻或赛博朋克
- [ ] 通用Dashboard卡片
- [ ] 大量重复模块
- [ ] 过度装饰
- [ ] 纯代码流程图
- [ ] 纯文字堆叠
- [ ] 其他：

### 信息密度

- [ ] 很低，强调极简
- [ ] 较低
- [ ] 中等
- [ ] 较高
- [ ] 很高，但必须可读

### 留白程度

- [ ] 大量留白
- [ ] 适度留白
- [ ] 紧凑布局
- [ ] 根据内容自动决定

### 视觉创新程度

- [ ] 保守，优先严谨和熟悉感
- [ ] 适度创新
- [ ] 明显创新，但不能改变论文逻辑
- [ ] 高度创新，允许实验性视觉表达

```text
例如：配色克制、主次明确、模块有真实语义、整体统一、局部细节丰富但不杂乱。
```

## 16. 建议提供量

### 零样本审美Profile

- 一段总体审美描述；
- 一份内容层级和重点突出规则；
- 3张正面参考图；
- 3张负面参考图。

### 少样本个性化Judge

- 10张以上正面参考图；
- 10张以上负面参考图；
- 20～100组A/B偏好；
- 每组一至三句判断理由；
- 明确的硬性拒绝条件。

### 完整个性化Judge

- 数百组以上偏好和生成轨迹；
- 多种论文和图片类型；
- 修改前图片、Judge建议、修改后图片；
- 建议是否有效的最终标签。
