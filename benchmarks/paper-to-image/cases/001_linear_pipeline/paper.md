# ExampleFlow: A Three-Stage Evidence Processing Framework

## Abstract

ExampleFlow transforms a Raw Document into a Final Prediction through Document Encoding and Evidence Aggregation.

## Method

The system takes a Raw Document as input. Document Encoding converts the Raw Document into contextual representations. Evidence Aggregation consumes the contextual representations and produces a Final Prediction.

The processing order is Raw Document, Document Encoding, Evidence Aggregation, and Final Prediction.
